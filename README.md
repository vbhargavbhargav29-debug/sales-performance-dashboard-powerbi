# sales-performance-dashboard-powerbi
# 📊 Next Shop Store Sales Performance Dashboard

## Project Overview

This project showcases an end-to-end Business Intelligence solution developed using Power BI to analyze retail sales performance. The dashboard transforms raw transactional data into meaningful insights through interactive visualizations, KPI tracking, and advanced DAX calculations.

The project demonstrates skills in data cleaning, data modeling, business reporting, and dashboard development, making it suitable for entry-level Data Analyst roles.

---

# Project Objectives

* Monitor overall sales performance.
* Analyze monthly revenue trends.
* Identify top-selling products.
* Compare Year-over-Year sales.
* Evaluate customer purchasing behavior.
* Track regional sales performance.
* Build an interactive dashboard for business decision-making.

# Data Preparation

Data preprocessing was completed in Power Query.

Steps included:

* Removed duplicate records
* Handled missing values
* Corrected data types
* Renamed columns
* Removed unnecessary columns
* Standardized text values
* Created calculated columns

---

# Data Modeling

Implemented a star schema consisting of:

* Fact Sales
* Calendar Table
* Product Table
* Customer Table
* Geography Table

Relationships were optimized for efficient reporting.
